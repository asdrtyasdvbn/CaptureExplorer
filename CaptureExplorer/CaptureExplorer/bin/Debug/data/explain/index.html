選取截圖-快速鍵
Esc 離開截圖模式
Ctrl + A 全選
Ctrl + C 複製
Ctrl + X 複製
Ctrl + D 取消選取
Ctrl + S 儲存圖片(png)
Enter 儲存圖片(png)
修改範圍時按著 空白鍵暫時變成拖曳
修改範圍時按著 Shift 正方形選取
滑鼠右鍵 取消選取 or 離開截圖模式
如何吸取顏色？
在局部截圖的模式下，直接對要吸取顏色的地方點一下，不進行拖曳，就會變成吸取顏色的工具
如何選擇儲存格式？
JPG 格式
檔案小、畫質普通
適合用於顏色很多的場合，例如：遊戲的截圖
PNG 格式
檔案大、畫質高
但顏色不多的話，檔案反而會比 JPG格式 更小
適合用於顏色較少的場合。例如：文件檔的截圖
截圖快速鍵 與 其他程式的快速鍵 衝突？
使用快速鍵進行截圖時，會取消原本按鍵的功能，因此可以放心的在任何按鍵上設定快速鍵

例如：
「F5」是瀏覽器的「重新整理」功能。
把截圖快速鍵設定成「F5」或「Ctrl + F5」，使用快速鍵進行截圖時，並不會觸發瀏覽器的重新整理功能
快速鍵完全沒有反應？
可能有其他權限更高的程式攔截鍵盤訊號，解決方法：
對程式按 右鍵→以系統管理員身份執行

注意：以系統管理員身份執行程式，將會無法使用拖曳至瀏覽器上傳圖片的功能
